B

Tabbed: Colony View + Code View + Comparison View

Three tabs at top. "Colony" shows one leader at a time (full width, rich detail). "Code" shows the TypeScript executing with syntax highlighting and live output. "Compare" shows the side-by-side split with HEXACO radar charts, tool registries, and colonist trajectories. User can switch between tabs as the simulation runs.
